Application

An application in the directory.

Example

http://ws-api.cloudaccess.net/ws/www/applications/:application_id

Fields

The Application object has the following fields.

Name Description Returns
id The application ID integer
title The application's title string
website The application's website string
description The application's description string
created_at The created date A struncture with date, timezone_type and timezone fields
modified_at The date when the application was modified last time A struncture with date, timezone_type and timezone fields
screenshots The application's screenshots Structure containing a data object and the count of total screenshots, with data containing an array of objects, each with the id and image_url to the screenshot
likes The application's likes Structure containing a data object and the count of total likes, with data containing an array of objects, each with the state and user_id of the like
hits The number of application's hits integer

GET

GET requests can be made to /applications in order to get all applications or to /applications/:application_id in order to get information about a specific applications.

Parameter Description Default
fields A string with the returned fields separated by comma. We recommend to request only the necessary fields in order to save computing time. All
since A string in date format (yyyy-MM-dd) representing the minimum created date 01-01-1970
before A string in date format (yyyy-MM-dd) representing the maximum created date NOW
offset An integer representing the first result 0
limit An integer representing the number of results
Note: The maximum results that can be returned are 100. 		20
user_id The user's ID of the returned applications -

If the request is successful, you get the following return.

Description Type Response Code
If request succeeded to /applications array of objects containing the requested fields 200
If request succeeded to /applications/:application_id Structure containing the requested fields 200

POST

POST requests can be made to /applications in order to create a new application or to /applications/:application_id to make PUT or DELETE requests if _method parameter exists in query.

Parameter Description Type Default Required
title The application's title string - yes
description The application's description string - yes
user_id The id of the user that creates the application integer - yes
website The application's website string null no
screenshots[] The application's screenshots multipart/form-data none no

If the create is successful, you get the following return.

Description Type Response Code
If the create succeeded Structure containing the id of the created application 200

PUT

PUT requests can be made to /applications/:application_id to update application's data. If your client does not support the PUT method you can use POST requests with the parameter _method=PUT.

Parameter Description Type
title The application's title string
description The application's description string
user_id The id of the user that created the application integer
website The application's website string

If the update is successful, you get the following return.

Description Type Response Code
If update succeded boolean 200

DELETE

DELETE requests can be made to /applications/:application_id to delete an application or to /applications to delete all applications. If your client does not support the DELETE method you can use POST requests with the parameter _method=DELETE.

Parameter Description Type
before The maximum created date date (yyyy-MM-dd)
since The maximum created date date (yyyy-MM-dd)
user_id The user_id of the aplication
Note: Works only for /applications requests 		integer

If the delete is successful, you get the following return.

Description Type Response Code
If delete succeded boolean 200

Connections

An Application object has the following connections.

Name Description Returns
comments All comments of this applicaton array of comment objects
tags All tags of this application array of tag objects
likes All likes of this application Structure containing a data object and the count of total likes, with data containing an array of objects, each with the state and user_id of the like.
screenshots All screenshots of this application Structure containing a data object and the count of total screenshots, with data containing an array of objects, each with the id and image_url of the screenshot.

Comments

Get all comments for this application
GET /applications/:application_id/comments
GET /comments?application_id=:application_id

Create a comment for this application
POST /applications/:application_id/comments
POST /comments?application_id=:application_id

Delete all application's comments
DELETE /applications/:application_id/comments
DELETE /comments?application_id=:application_id

Tags

Get all tags for this application
GET /applications/:application_id/tags
GET /tags?application_id=:application_id

Add a tag to this application
POST /applications/:application_id/tags?name=tag
POST /tags?application_id=:application_id&name=tag

Add a list of tags to this application
POST /applications/:application_id/tags?list=tag1,tag2,tag3
POST /tags?application_id=:application_id&list=tag1,tag2,tag3

Add a list of tag_id to this application
POST /applications/:application_id/tags?ids=tag_id1,tag_id2,tag_id3
POST /tags?application_id=:application_id&ids=tag_id1,tag_id2,tag_id3

Remove a tag from this application
DELETE /applications/:application_id/tags?name=tag
DELETE /tags?application_id=:application_id&name=tag

Remove a list of tags from this application
DELETE /applications/:application_id/tags?list=tag1,tag2,tag3
DELETE /tags?application_id=:application_id&list=tag1,tag2,tag3

Remove a list of tag_id from this application
DELETE /applications/:application_id/tags?ids=tag_id1,tag_id2,tag_id3
DELETE /tags?application_id=:application_id&ids=tag_id1,tag_id2,tag_id3

Remove all tags of this application
DELETE /applications/:application_id/tags
DELETE /tags?application_id=:application_id

Likes

Get all likes for this application
GET /applications/:application_id/likes
GET /likes?application_id=:application_id

Like this application
PUT /applications/:application_id/like

Unlike this application
PUT /applications/:application_id/unlike

Screenshots

Get all screenshots for this application
GET /applications/:application_id/screenshots
GET /screenshots?application_id=:application_id

Add a screenshot to this application (screenshots[] parameter as multipart/form-data)
POST /applications/:application_id/screenshots
POST /screenshots?applications_id=:application_id

Delete a screenshot from this application
DELETE /applications/:application_id/screenshots/:screenshot_id
DELETE /screenshots/:screenshot_id?application_id=:application_id

Delete more screenshots from this application
DELETE /applications/:application_id/screenshots?ids=ss_id1,ss_id2,ss_id3
DELETE /screenshots?application_id=:application_id&ids=ss_id1,ss_id2,ss_id3

Delete all screenshots of this application
DELETE /applications/:application_id/screenshots
DELETE /screenshots?application_id=:application_id
